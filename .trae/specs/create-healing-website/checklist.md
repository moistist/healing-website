# Checklist

## 后端功能验证
- [ ] FastAPI 服务可正常启动，CORS 配置正确
- [ ] SQLite 数据库自动创建，表结构正确
- [ ] 环境变量配置读取正常（API Key、Base URL、模型名称）
- [ ] AI 对话接口（/api/chat）可正常调用，支持普通和流式响应
- [ ] 对话历史记录保存和查询功能正常
- [ ] 情绪记录接口（/api/emotions）可正常创建和查询
- [ ] 情绪趋势统计接口返回正确数据
- [ ] 心情日记接口（/api/journal）支持创建、查询、删除
- [ ] 日记列表支持分页和日期筛选

## 前端功能验证
- [ ] React 项目可正常启动，路由配置正确
- [ ] 全局样式符合心理疗愈风格（柔和色调、圆润边角、舒适字体）
- [ ] 响应式布局在桌面和移动端正常显示
- [ ] AI 对话页面可发送消息并接收回复，流式响应正常显示
- [ ] 情绪记录页面可选择情绪并提交记录
- [ ] 情绪历史列表和趋势图表正常展示
- [ ] 冥想/呼吸练习页面动画引导正常，计时器功能正确
- [ ] 心情日记页面可创建、查看、删除日记
- [ ] 日记列表按日期分组展示，筛选功能正常
- [ ] 首页布局美观，功能入口导航正常

## 前后端联调验证
- [ ] 前端 API 请求正确转发至后端
- [ ] 对话功能（普通和流式）联调成功
- [ ] 情绪记录和日记功能联调成功
- [ ] 数据持久化正常（刷新页面后数据保留）

## 部署配置验证
- [ ] 环境变量配置文件示例（.env.example）存在且内容正确
- [ ] 启动脚本可正常运行后端和前端
- [ ] 生产环境构建配置正确
